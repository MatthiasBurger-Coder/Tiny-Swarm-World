= Deployment View

Deployment strategy and distribution.

== Local Runtime State

Tiny Swarm World remains a single local Python automation application for this
workflow slice. Desired inventory belongs to committed configuration and may be
stored under `infra/config`.

Observed inventory and verification evidence are deployment-local runtime
artifacts. They are stored under the ignored `.tiny-swarm-world/` directory in
the repository root. That directory is not a deployment input, is not part of
the default quality gate outputs, and must not be committed or moved under
`infra/config`.
