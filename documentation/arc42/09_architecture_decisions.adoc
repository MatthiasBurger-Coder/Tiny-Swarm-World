== Architectural Decisions

This section includes the active ADRs from `documentation/arc42/09_decisions/` in
the assembled arc42 book. Standalone ADR files keep their own history and
status; this chapter groups and includes them without renumbering or rewriting
ADR history.

In the assembled `documentation/arc42.adoc` book this section is rendered as
chapter 10 because the book starts with a generated system overview chapter
before the maintained arc42 section files.

=== ADR Location Convention

Standalone ADR files currently live under `documentation/arc42/09_decisions/`
with the naming pattern `adr-<decision-slug>.adoc`.

`documentation/arc42/09_architecture_decisions.adoc` is the active arc42 entry
point for those decisions. Existing ADR files must not be silently renumbered
or renamed as part of documentation consolidation or workflow execution.

The repository does not currently use `documentation/adr/**`. Introducing that
path requires a separate convention decision, including
`documentation/adr/README.md`, reference migration, and history-preserving
links from existing ADR files.
