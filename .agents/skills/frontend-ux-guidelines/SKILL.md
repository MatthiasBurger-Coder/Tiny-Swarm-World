# UX Guidelines

## Purpose

Guide user experience for forensic analysis, replay, reporting and evidence review.

## Practices

- Optimize for scanning, comparison and repeated investigation.
- Make evidence status visible without relying on color alone.
- Separate confirmed facts, derived analysis, unresolved gaps, hypotheses and suggested fixes.
- Use restrained, task-focused layouts for operational tools.
- Keep visualization deterministic and traceable to source data.

## Verification

- Review accessibility, responsive layout and evidence labels.
- Validate screenshots or UI tests when frontend behavior changes.
